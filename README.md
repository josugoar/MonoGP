## Usage

```sh
# Clone MMDetection3D
git clone https://github.com/open-mmlab/mmdetection3d.git
# Clone MonoGP to the correctly named directory in the projects folder
git clone https://github.com/josugoar/monogp.git projects/MonoGP
```
